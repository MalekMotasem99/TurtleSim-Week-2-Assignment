Description :

This repository contains my Week 2 assignment for the TurtleSim Mastery task in the ROS2 course.  
It demonstrates the creation and execution of a custom package called **robot_controller**, which includes multiple nodes to publish and subscribe to topics.

The main tasks demonstrated in this project are:  
  Creating and building a custom ROS2 package  
  Writing publisher and subscriber nodes  
  Running and testing all nodes successfully  
  Observing message exchange between nodes
